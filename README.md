# Todo-App
## _A Simple Todo-app written in Next.js + ExpressJS + MongoDB_
## Get Started
1. Clone the repo
`git clone https://github.com/xAbdaziz/todo-app.git`
2. Install the dependencies
```sh
cd frontend
yarn install
cd ../backend
yarn install
```

3. Run the code

Frontend:
```sh
yarn dev
```
Backend:
```sh
yarn start
```
